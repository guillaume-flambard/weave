Weave — Cognitive Runtime. Weave is a shared organizational memory layer for
knowledge-work teams: it turns everyday AI activity (questions, answers, decisions
across Slack, docs, PRs) into structured memory, and makes reusable **skills** and
specialist **agents** *emerge automatically* from recurring patterns — nobody writes
them by hand. Memory is scoped across four levels: personal → team → project →
organization, and every answer is traceable back to the sources that produced it.

Surfaces: a web dashboard/console (live activity feed, memory browser, emergent-skill
and agent panels, an "ask the org" interface) plus an in-product guided tour. Audience:
product, ops, finance, engineering and growth teams inside modern SaaS companies
(design partners: PennyLane, Notion). It should feel like a calm, precise,
Notion-grade tool — not a flashy AI demo.

Design system name: **Weave Design System**.
